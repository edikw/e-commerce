# e-commerce with framework vue.js and vuetify

## Project setup
```
change directori -> e-commerce
change directori -> my-layout
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
